# MODULE_MAP — Hedef Domain / Modül Haritası (Geçici)

Son güncelleme: 2026-07-17 (F0-001)

> **UYARI:** Bu harita, program bağlamından türetilmiş **geçici bir hedef-domain haritasıdır**. Bu görevde (F0-001) tam depo modül denetimi **yapılmamıştır**. Aşağıdaki **her girdi** şu iki etiketi taşır:
>
> - **TARGET DOMAIN** (hedef alan; mevcut kod yapısını değil, hedeflenen modül sınırını ifade eder)
> - **NOT YET REPOSITORY-VERIFIED** (henüz depo kanıtıyla doğrulanmamıştır)
>
> **F0-003**, gerçek dosya sahipliğini doğrulayacak ve bu haritayı revize edebilecektir.
